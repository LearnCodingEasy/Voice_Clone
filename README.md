# Voice_Clone

My Voice Clone

### Create a virtual environment

```
python -m venv venv
```

### Activate a virtual environment

```
venv\Scripts\activate
```

### Download Coqui TTS

```
git clone https://github.com/coqui-ai/TTS.git
```

```
cd TTS
```

```
pip install -r requirements.txt
```

```
pip install -e .
```

```
pip install soundfile
```
